# rapidLAB - Laboratory Logistics Demo

**rapidLAB** is a lightweight, conceptual system developed to manage and track the internal transport of laboratory samples and materials efficiently.

⚠️ *Note*: This repository hosts a basic web demo for demonstration purposes.

## 🌐 Live Preview

Explore the live deployment here: [Live Demo](https://juan-a-moralesh.github.io/demoLab/)

## 🛠️ Core Features (Demo)

* Basic sample tracking and status updates.
* Material logistics management.
* Clean and intuitive interface designed for fast laboratory workflows.
